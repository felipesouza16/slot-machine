## Stack utilizada

**Front-end:** Vite, Phaser

## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/felipesouza16/slot-machine.git
```

Entre no diretório do projeto

```bash
  cd slot-machine
```

Instale as dependências

```bash
  npm install || yarn install
```

Inicie o servidor

```bash
  npm run dev || yarn dev
```
